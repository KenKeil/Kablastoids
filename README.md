# Kablastoids
Python Space Shooter Game

To install:
  - Place all files and folders inside a created folder
  - Run Space.py
  
  Controls:

    A - move Left
  
    D - move right
  
    W - Shoot
  
    Space - Shoot
  
    Shift - Boost
  
    Q - Quit
  
  For Gamepad:
  
    Left - move left
    
    Right - move right
    
    Up - shoot
    
    Left bumper - boost
    
    Right bumper - boost
    
    A - shoot
    
    B - shoot
    
    X - shoot
    
    Y - shoot
    
    Start - Quit
